讲一个科学上网的故事

Aug 23 2017


自从今年天朝开始严打翻墙之后，我就不能忍了，作为二十一世纪祖国的花朵，怎么就不能上Google看Youtube呢？

于是乎…

我用了很多方法，有免费的，有付费的，但都不是很耐用。

所以…

我拉上我的搭档 PR 打算自己搭梯子…

www.tmdvpn.com 诞生了，这个网站是在今年初开始正式上线。从网站功能建设，网站设计、教程设计以及整体框架设计，前后经历了快一年的迭代(下面是一些版本更新记录，这基本都是用睡觉的时间在做)。

下面是网站截图
www.tmdvpn.com

最近发生很多事，国内封杀各种翻墙方法，苹果商店近一个月基本所有大家知道的vpn客户端都被下架，我们的tmdvpn没有被查水表，我们也不会再去宣传，都是熟人之间用。

最后要说一下的是，我们之所以做了TMDVPN，因为我们相信，每个人，不管他们的技术经验水平如何，都应该能够自由和安全的使用互联网。能够真正利用互联网获取对自己有用的东西。
